# Terminal Helper

A simple C utility built to automate and simplify terminal usage.

## Purpose
This project serves two main goals:
1. **Practical Tooling:** A lightweight helper for my day-to-day workflow in the terminal.
2. **Learning Lab:** My personal space for studying pure C and moving away from high-level abstractions to understand systems programming better.

## Usage

### Compilation
To build the binary on Linux or macOS:
```bash
gcc terminal-helper.c -o terminal-helper
```

### Execution
Search for a specific keyword in your history:
```bash
./terminal-helper <search_term>
```
